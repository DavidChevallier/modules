## Introduction

A library of string validators and sanitizers like [validator.js](https://github.com/validatorjs/validator.js)

## How to Use

```shell
kcl mod add validator
```

```python
import validator

is_alpha_str = validator.is_alpha("ccc", locale='en-US', options=None)
```
